# Redisseny d’un Servei Digital Municipal

## Context

L’Ajuntament del teu poble vol renovar un dels seus serveis digitals perquè actualment presenta problemes d’usabilitat, estructura confusa i disseny antiquat. Per això ha contractat un equip de desenvolupadors i dissenyadors… vosaltres.

La vostra missió és analitzar, planificar, dissenyar i implementar una proposta funcional d’una part d’aquest servei digital, aplicant tots els coneixements vists fins ara en l'assignatura de Disseny d'Interfícies Web.

## Servei a redissenyar

L’Ajuntament vos permet escollir entre una d’aquestes seccions:

* Reserva d’equipaments municipals (sales, pistes esportives, aules d’estudi…).
* Portal de consultes i queixes del ciutadà.
* Informació d’activitats culturals i d’esdeveniments (agenda municipal).
* Portal de transparència econòmica per a visualitzar la relació trimestral de contractes menors.
* Portal d'incidències municipals.
* Millora de la visualització i consulta de les ordenances municipals que actualment es mostren en format pdf. Integració d'un chatbot.

> [!NOTE]
>
> El contingut ha de ser inventat però creïble (nom d’equipaments, activitats, fotos lliures de drets, etc.).

## Objectiu

Crear una versió redissenyada d’una secció del web municipal, incorporant els principis bàsics del disseny d’interfícies i implementant-la mitjançant HTML, CSS, CSS avançat i tècniques modernes de layout.

El resultat final inclourà:

* Dossier de disseny
* Wireframes, mockups i prototip
* Memòria del treball fet
* Presentació del projecte

## Fases

### Fase 1

* Dossier de disseny
* Mapes, esbossos, wireframes, mockups, prototip

### Fase 2

* Implementació HTML
* CSS, CSS avançat, Layout

### Fase 3

* Exposició d'uns 10 minuts de la tasca i les funcionalitats desenvolupades.

### Entregables

* Dossier
* Projecte HTML/CSS
* Memòria tècnica
* Presentació
